# Dr. Peter Waweru Website

Official professional website for **Dr. Peter Kuria Waweru**, a Vascular & Endovascular Neurosurgeon, Stroke Specialist, Researcher, and Advocate dedicated to advancing stroke care, neurovascular health, and health equity across Kenya and Africa.

## Overview

This website serves as a central platform for:

- Professional profile and biography
- Clinical expertise and areas of specialization
- Research portfolio
- Academic publications
- Stroke awareness and public education
- Media appearances and speaking engagements
- Research collaborations and partnerships
- Professional inquiries and consultations

## Features

- Modern responsive design
- SEO optimized pages
- Structured data (Schema.org)
- Open Graph and social sharing support
- XML Sitemap
- Robots.txt configuration
- Mobile-friendly user interface
- Professional academic and healthcare branding
- WhatsApp-powered contact inquiries
- Smooth animations with Framer Motion

## Local Development

Clone the repository:

```bash
git clone https://github.com/KennedyMaurice/PeterWaweru.git
```

Navigate to the project:

```bash
cd PeterWaweru
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## Production Build

Create production build:

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```

Generate ZIP deployment package:

```bash
npm run build:zip
```
